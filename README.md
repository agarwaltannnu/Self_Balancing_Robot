# 🤖 Self-Balancing Robot (PID Control)

A two-wheeled self-balancing robot built using Arduino and MPU6050 IMU sensor.  
This project demonstrates real-time feedback control using a PID (Proportional-Integral-Derivative) algorithm.

---

## 🚀 Overview

The robot maintains balance by continuously measuring its tilt angle and adjusting motor speed accordingly using a control loop.

---

## ⚙️ Tech Stack

- Arduino UNO  
- MPU6050 (Accelerometer + Gyroscope)  
- L293D Motor Driver  
- DC Motors  
- Arduino IDE (Embedded C)  

---

## 🧠 Working

- MPU6050 provides tilt data  
- Arduino computes error (desired angle vs current angle)  
- PID controller calculates correction  
- Motor driver adjusts motor speed using PWM  
- Robot stabilizes in real-time  

---

## ⚡ Key Features

- Real-time feedback control  
- PID-based stabilization  
- Hardware-software integration  
- Compact robotic system  

---

## 🛠️ Challenges
